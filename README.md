# مشروع الويب التفاعلي

موقع متعدد الصفحات متجاوب مبني بـ **HTML5** و **CSS3** و **Bootstrap 5** و **JavaScript** (Vanilla)، بواجهة عربية (RTL) وتصميم رسمي مع وضع ليلي افتراضي.

## الصفحات
- **index.html** — الصفحة الرئيسية وروابط الأقسام.
- **blog.html** — مدونة (35 مقالًا) مع ترقيم صفحات (Pagination).
- **post.html** — صفحة مقال مفرد تُحمَّل ديناميكيًا عبر `?id=`.
- **gallery.html** — معرض صور مع فلترة حسب التصنيف.
- **resume.html** — السيرة الذاتية مع روابط المشاريع وتحميل PDF.

## البنية
```
├── *.html              # صفحات الموقع
├── css/
│   ├── base.css        # تنسيقات مشتركة + الوضع الليلي
│   ├── home.css · blog.css · post.css · gallery.css · resume.css
└── js/
    ├── common.js       # الوضع الليلي (مشترك)
    ├── data.js         # بيانات المقالات الثابتة
    └── blog.js · post.js · gallery.js · resume.js
```

## المزايا
- تصميم متجاوب كامل (جوال / تابلت / حاسوب).
- وضع ليلي افتراضي مع حفظ التفضيل في `localStorage`.
- بيانات المدونة في ملف منفصل + ترقيم صفحات.
- كل صفحة لها ملف CSS و JS خاص بها.

## التشغيل
يُفضّل فتح المشروع عبر خادم محلي (مثل Live Server) لأن صفحات المدونة تقرأ البيانات عبر JavaScript.
